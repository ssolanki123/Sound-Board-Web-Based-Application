Script & Noscript

We rarely see a static website nowadays thanks to the rapid growth of Javascript. Javascript makes a website more interactive. We can have Javascript embedded into our HTML page by using the built-in <script> tag. For example

Click to change the message


The message changed because of Javascript. To do this, we have to insert a Javascript code at the bottom of the document.

   <script>
   function clicky(){
      document.getElementById("msg").innerHTML = "Hello from Javascript!";
   };
   </script>

Some old websites may not be able to run Javascript. To let the user know that Javascript is needed in their browser, we use <noscript> tag.

   <noscript>Javascript is turned off!</noscript>

Style & Link

To have a pure HTML page is quite boring. This is where style tag comes to the picture. Style tag lets you decorate your site, typically by changing the font, background, font-color etc. For example:

Wouldn't it be fantastic to have some style in your page?

There are 2 methods to have the text in your page be formatted that way. Firstly, there's an inline method of including stylesheet into HTML.

<p style="color" a color:>Hello, this text will be colored Red.</p>

Another method is to have a separate CSS file Linked to the html page via a link.

<link rel="stylesheet" type="text/css" href="file.css">

Select, Opt, & Optgroup

There are times when you want the user to select a specific option. This can be achieved through the use of a drop-down list. To create a drop-down list, HTML provides a select tag along with the accompanying opt and optgroup tag to provide the available option to the end user. For example:

Please select which pizza variation you would like to order:


To use the Select, Opt, and Optgroup tag in the HTML, type in

   <select>
      <optgroup label="some_label">
         <opt>some option>
      </optgroup>
   </select>

Anchor

Anchor element in HTML lets you create hyperlink in a HTML object.


To achieve this result, use the semantics of anchor element as per following:

   <a href="some_link">insert text or image here</a>


Ins & Del

Sometimes you want to emphasize a part of a text that's been inserted/deleted into/from a page. We can do this via the ins and del HTML tag. An example of the use of ins and del tag is shown below:

I would like to inform you that some of the word has been inserted deleted

The semantic for these two are pretty straightforward.

   <ins>The inserted text</ins>
OR
   <del>The deleted text</del>


List

To list items in HTML, we use the tag ul for unordered list and ol for ordered list. In addition, every item must be enclosed within the li tag.


This is an unordered list:
  • First item
  • Second item

This is an ordered list:
  1. First item
  2. Second item

The semantic for both is shown below:

   <ul or ol>
      <li>item 1 </li>
      <li>item 2 </li>
      etc
   </ul or ol>

There is also a description list to put item's description. The description list has the same feel to regular list. There are tags that accompany one another which are dl, dt, and dd.

Internet
The most awesome invention.
Memes

The semantics for these desriptions lists are

   <dl>
      <dt>Item to describe</dt>
         <dd>Description</>
   <dl>


iframe

The iframe tag is used to embed an independent HTML document into the current document.


The semantic for iframe is as follows

   <iframe src="Other HTML document goes here"></iframe>
